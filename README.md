## 명성아이지
중문 생산 및 판매하는 명성아이지 기업의 제품소개 사이트입니다.<br>
[https://www.igdoor.co.kr](https://www.igdoor.co.kr/main/main.html)

## 작업 의도
화면 전체 너비를 활용한 불규칙적인 레이아웃의 느낌을 디바이스 별로 유지하는데 주력했으며<br>
Tooltip 플러그인을 활용하여 제품의 소재를 확대해서 볼 수 있도록 구현했습니다.
